# **App Name**: Nexus-Med: AI Medication Insights

## Core Features:

- Medication List Analysis: Analyze a user's medication list against a database of known drug interactions.
- Personal Health Context Integration: Incorporate patient's age, allergies, and diagnoses to filter relevant interactions.
- Food Query Analysis: Identify potential interactions between medications and user-specified foods.
- AI-Powered Interaction Report: Synthesize data using DrugBank, ML models, and the Gemini API to provide an easy-to-understand, AI-generated report with contextual alerts using a tool.
- AI Chatbot Integration: Enable users to ask follow-up questions regarding the medication interaction report.
- API Access: Allow third party apps and developers to use Nexus-Med features using interactive API documentation and code samples
- Wallet Medication Card: Offer premium users the ability to store their prescriptions and allergy information on the Nexus-Med digital wallet for mobile use.

## Style Guidelines:

- Background: Very dark gray (#1F1F1F) to provide a high-contrast, immersive dark mode experience.
- Foreground: Off-white (#E0E0E0) for high-contrast text, ensuring readability.
- Card: Slightly lighter dark gray (#2C2C2C) for card backgrounds, providing subtle separation.
- Border: Soft, subtle gray (#333333) for borders and separators, maintaining a clean look.
- Primary (Brand/Action): Vibrant Teal/Cyan (#53D8C4) for primary buttons, links, and chart data, conveying trustworthiness.
- Accent (Warning/Secondary CTA): Bright Orange (#FFA540) for warnings and secondary buttons, drawing attention to critical information.
- Headlines (H1, H2, Titles): 'Bricolage Grotesque' sans-serif for a distinctive and modern look.
- Body (Paragraphs, Labels, UI Text): 'DM Sans' sans-serif for clean and legible UI text.
- Code samples (API): 'Source Code Pro' monospace
- Cards: Use bg-card, subtle border-border, and rounded corners (rounded-lg) for a modern layout.
- Glassmorphism: Apply a semi-transparent, blurred "frosted glass" background (bg-card/80 backdrop-blur-sm) to the header and chat interface.
- Glow & Pulse: Key CTAs should have a soft, inviting Orange glow and slow pulse.
- Text Reveal: Hero text and taglines should use a smooth "blur-in" animation.
- Interactive Cards: Feature cards on the homepage should have a subtle 3D tilt-on-hover effect.
- Teal color is used for pledge icons, and orange color is used for warnings.
